# NVDA Markdown 导航

Markdown Navigator 是一个 NVDA 插件，旨在提升编辑和阅读 Markdown 内容时的体验。它允许你在可编辑的文本区域（如记事本、VS Code、网页编辑器等）中使用类似于 NVDA 浏览模式的单键导航功能。

它可以让你无需频繁按方向键就可以快速跳转到标题、列表、表格、代码块以及各种行内格式。

## 主要特性

*   高效导航：实现了快速算法，即使在数万行的大型文档中也能秒级跳转。
*   结构化浏览：支持按标题、列表、表格、引用块、代码块等结构跳转。
*   行内元素：支持跳转到粗体、斜体、链接、图片和行内代码。
*   表格单元格导航：提供与  NVDA 一致的快捷键在 Markdown 表格中按单元格移动。


## 使用方法

### 切换浏览模式
默认情况下，单键导航是关闭的。你需要手动开启 Markdown 浏览模式。

*   快捷键：NVDA + Shift + Space (空格键)
*   开启或关闭插件功能。NVDA 会朗读 "Markdown Browse Mode On/Off"，该功能也将跟随 NVDA 浏览模式中“用音效提示焦点模式或浏览模式”设置。

### 快捷键列表 (浏览模式开启时)

当模式开启后，你可以使用以下单字母按键：

*   按以下字母键 跳转到下一个元素。
*   按 Shift + 对应字母键跳转到上一个元素。

| 按键 | 元素类型 | 说明 |
| --- | --- | --- |
| H | 标题 | 任意级别的标题 (# 到 ######) |
| 1-6 | 1-6级标题 | 跳转到特定级别的标题 |
| T | 表格 | 表格起始处 |
| L | 列表 | 列表块的开始 |
| I | 列表项 | 具体的列表项 (-, *, 1.) |
| Q | 引用块 | 引用段落 (>) |
| C | 代码 | 代码块 (```) 或行内代码 (`) |
| S | 分隔线 | 水平分割线 (---, ***) |
| X | 复选框 | 任务列表项 (- [ ], - [x]) |
| K | 链接 | Markdown 链接 [文本](链接) |
| G | 图片 (Graphic) | 图片标签 ![说明](链接) |
| B | 粗体 (Bold) | 粗体文本 (** 或 __) |
| E | 斜体 (Emphasis) | 斜体/强调文本 (* 或 _) |
| D | 删除线 (Delete) | 删除线文本 (~~) |
| F | 脚注 (Footnote) | 脚注引用 [^1] |
| , | 块结尾 | 跳转到当前块级元素的末尾 |
| Shift+, | 块开始 | 跳转到当前块级元素的开头 |

### 表格单元格导航

当光标位于 Markdown 表格内时，若开启了浏览模式，可以使用以下快捷键在单元格间移动：

*   Ctrl + Alt + 左光标: 上一个单元格
*   Ctrl + Alt + 右光标: 下一个单元格
*   Ctrl + Alt + 上光标: 上一行同列
*   Ctrl + Alt + 下光标: 下一行同列


## 注意事项

*   本插件专为 markdown 编辑场景而开发。如果你正在阅读浏览器渲染后的 HTML 页面，请使用 NVDA 的浏览模式。
*   如果按下上述按键是键入字符而不是位置跳转，请检查是否按下了 NVDA + Shift + Space 开启了浏览模式。
*     在 VS Code 中，由于编辑区域一次性呈现的内容有限，该插件的浏览模式可能不符合预期。
